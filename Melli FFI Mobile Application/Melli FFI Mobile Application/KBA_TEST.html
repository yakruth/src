AD Domain Administrative Tools for Win7

 

1.    Download the x86fre_GRMRSAT_MSU.msu (32 bits)

http://www.microsoft.com/downloads/en/details.aspx?FamilyID=7d2f6ad7-656b-4313-a005-4e344e43997d&displaylang=en

2.    Login with local admin account.

3.    Double click the file x86fre_GRMRSAT_MSU.msu to unpack it, and then start the Remote Server Administration Tools for Windows 7 Setup Wizard.
Important: You must accept the License Terms and Limited Warranty to start to install the Administration Tools pack.

4.    Complete all the steps that you must follow by the wizard, and then click Finish to exit the wizard when installation is completed.

5.    Click Start, click Control Panel, and then click Programs.

6.    In the Programs and Features area, click Turn Windows features on or off.

7.    In the Windows Features dialog box, expand Remote Server Administration Tools.

8.    Expand Role Administrator Tools.

9.    Expand AD DS and AD LDS Tools.

10. Expand AD DS Tools.

11.  Check Active Directory Administrative Center and AD DS Snap-ins and Command-line Tools. Then click OK.

 


12. Configure the Start menu to display the Administration Tools shortcut, if it is not already there.
Right-click Start, and then click Properties.
On the Start Menu tab, click Customize.
In the Customize Start Menu dialog box, scroll down to System Administrative Tools, and then select Display on the All Programs menu and the Start menu. Click OK. Shortcuts for snap-ins installed by Remote Server Administration Tools for Windows 7 are added to the Administrative Tools list on the Start menu.

 

13. Open Control Panel\All Control Panel Items\Administrative Tools\Active Directory Users and Computers
